About Me!

Halo, saya adalah seorang siswa SMK Taruna Bhakti jurusan Pengembangan Perangkat Lunak dan Gim (PPLG). Saya sudah mempelajari dasar-dasar pengembangan web, termasuk HTML, CSS, dan JavaScript. Dengan bekal ilmu tersebut, saya terus mengasah keterampilan saya dalam membuat website yang fungsional dan menarik.

Saya sangat bersemangat untuk belajar lebih banyak, terutama tentang teknologi terbaru di bidang pemrograman dan pengembangan web. Selain itu, saya juga berencana untuk mencari pengalaman melalui magang atau proyek kolaborasi, sehingga bisa memahami kebutuhan industri sekaligus mengembangkan kemampuan saya lebih lanjut.

Saya percaya bahwa kombinasi antara ilmu yang saya pelajari dan semangat saya untuk terus berkembang akan menjadi bekal penting untuk meraih masa depan yang sukses di dunia teknologi.
